About This FYP

The objective of this project is to develop a web application running on the cloud. It will showcase the capabilities of the cloud and how easy it is for anyone to develop a web application for the cloud. By harnessing the power of cloud computing, the project will show that one no longer needs to own or manage a server and other required setups to run a fully capable application that can serve hundreds or thousands.
