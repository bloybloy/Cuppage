Application Requirements

  1. User Account Management

    The user will be able to change his password and update his profile.

  2. Dashboard

    This will be the userâ€™s main view. All users will share the same view so that every team member is on the same page in terms of the current progress of the project and what is happening next. This view will provide a visual for the overall progress.

  3. Create, edit and assign tasks

    From the dashboard, the user will also be able to create a task. Users with a team leader role can assign tasks to other users.

  4. Accept, reject and give feedback for tasks

    A user can choose to accept or reject an assigned task. He can also give feedback for a task assigned to him or others. For example, when a team member feels that he can handle a particular task better, he can request the team leader to reassign the task through feedback.
