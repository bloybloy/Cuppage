Cuppage: An Application In Development

The application to be developed will be an easy to use project management tool that will help bring focused collaboration to the team. To provide some context for this web application and help this project be more focused, our end users in this project will be a team of event planners. Their work revolves around tight deadlines, is highly task-oriented and heavily dependent on effective information sharing.


The final product is a simple and powerful web application for event planners, yet versatile enough to be extended to other types of users.
